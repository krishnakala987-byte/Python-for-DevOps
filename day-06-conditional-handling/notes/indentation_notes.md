# Indentation Notes

## What is Indentation?

Indentation means spacing before code blocks.

Python uses indentation to identify:
- condition blocks
- loops
- functions

---

## Correct Example

```python
if cpu > 80:
    print("Critical Alert")
```

---

## Incorrect Example

```python
if cpu > 80:
print("Critical Alert")
```

---

## Why Important?

Incorrect indentation causes:
- IndentationError
- invalid block structure
