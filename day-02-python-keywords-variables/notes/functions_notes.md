# Python Functions Notes

## What is a Function?

A function is a reusable block of code.

Example:

```python
def greet():
    print("Hello")
```

---

## Why Functions are Important

Functions:
- reduce repetition
- improve readability
- make scripts reusable
- simplify maintenance

---

## Function with Parameters

```python
def greet(name):
    print(f"Hello {name}")
```
