# Python Keywords Notes

## What are Keywords?

Keywords are reserved words in Python that have special meanings.

Examples:
- if
- else
- elif
- for
- while
- def
- try
- except
- return
- import

---

## Why Keywords are Important

Keywords control:
- decisions
- loops
- functions
- error handling
- program flow
