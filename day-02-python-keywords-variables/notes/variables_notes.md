# Python Variables Notes

## What is a Variable?

A variable stores data.

Example:

```python
server_name = "web1"
```

---

## Why Variables are Important

Variables:
- avoid hardcoding
- make scripts dynamic
- improve maintainability
- reduce human errors

---

## Types of Variables

### String

```python
name = "Krishna"
```

### Integer

```python
cpu = 90
```

### Float

```python
memory = 75.5
```

### Boolean

```python
server_running = True
```

### List

```python
servers = ["web1", "web2", "web3"]
```
