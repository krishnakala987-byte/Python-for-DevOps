# Data Types in Python

Python uses different data types to store different kinds of information.

## Common Data Types

### String

Used for text values.

Example:

```python
name = "Krishna"
```

### Integer

Used for whole numbers.

```python
age = 22
```

### Float

Used for decimal numbers.

```python
cpu_usage = 72.5
```

### Boolean

Used for True or False values.

```python
server_running = True
```

## Dynamic Typing

Python automatically detects data types.

```python
name = "DevOps"
```

No need to define datatype manually.
