# String Functions

## upper()

Converts string to uppercase.

```python
text = "devops"

print(text.upper())
```

## lower()

Converts string to lowercase.

```python
print(text.lower())
```

## split()

Splits string into parts.

```python
arn = "arn:aws:iam::123456789:user/krishna"

print(arn.split("/"))
```

## replace()

Replaces text.

```python
text = "hello"

print(text.replace("h", "H"))
```
