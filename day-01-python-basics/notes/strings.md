# Strings in Python

A string is a sequence of characters.

Example:

```python
text = "devops"
```

## String Indexing

```python
print(text[0])
```

Output:

```text
d
```

## Negative Indexing

```python
print(text[-1])
```

Output:

```text
s
```

## String Slicing

```python
print(text[0:3])
```

Output:

```text
dev
```
