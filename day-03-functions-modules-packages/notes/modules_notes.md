# Modules Notes

## What is a Module?

A module is a Python file containing reusable code.

Example:

```python
import calculator
```

---

## Why Modules are Important

Modules:
- prevent code duplication
- improve organization
- support scalability

---

## Example Module

```python
def add(a, b):
    return a + b
```

---

## DevOps Use Cases

- AWS utility modules
- monitoring utilities
- deployment scripts
- logging systems
