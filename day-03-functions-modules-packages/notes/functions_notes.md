# Functions Notes

## What is a Function?

A function is a reusable block of code.

Example:

```python
def greet(name):
    print(f"Hello {name}")
```

---

## Why Functions are Important

Functions:
- reduce repetition
- improve readability
- simplify debugging
- improve maintainability

---

## Parameters

Parameters receive input dynamically.

Example:

```python
def add(a, b):
    return a + b
```

---

## Return Keyword

Returns output from a function.

Example:

```python
def multiply(a, b):
    return a * b
```
